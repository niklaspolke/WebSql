# WebSql
Web Application to check or modify the contents of sql databases

### Features:
* display configured jdbc resources on tomcat
* execute sql commands

### Architecture:
* Java EE
* JSP / EL / JSTL
* JNDI / JDBC / HSQLDB
* SLF4J
* Maven
* Tomcat
